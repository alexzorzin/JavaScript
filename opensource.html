What is an Open Source Software?

An Open Source Software is a free program whose source code can be used, studied and copied for any purpose. Many people redistributes programs with or without changes. Some examples of this type of Software is the "Chromium" browser, based on Google Chrome, since it offers its source code for free and new browsers came from there.
